Bootstrap Grid System

Bootstrap

Bootstrap is a front-end framework for developing responsive web applications. It specializes in mobile-first designs using HTML, CSS, and Javascript. The Bootstrap Grid System offers a simplified way to position and size your page's content through the use of pre-set classes that divide the page into rows of 12 columns.
